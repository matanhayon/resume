Finding Bridges in Undirected Graphs

This project is a C++ program that examines an undirected graph to determine if it's connected and, if so, identifies all the bridges within the graph. It accomplishes this by implementing graph algorithms to analyze the connectivity of the graph and find critical bridges, which can be important for network design and analysis.

GitHub Repository: Bridges Algorithm

Eulerian Graph Checker

This C++ program is designed to determine if a given graph is Eulerian, and if it is, it finds and prints the Eulerian circle within the graph. The program prompts the user to specify whether the graph is directed or not, the number of vertices, and the number of arcs. It then reads the pairs of arcs and applies graph algorithms to determine Eulerian properties.

GitHub Repository: Euler Graph Checker
